BT-JS

BT-JS (BlenderTimer Javascript) is a free javascript library which includes many pre-built functions to make certain web programming areas easier, quicker, and just more time efficient! With BT-JS you can quickly get a random item from a list, get the cursor position relative to an element, convert text to proper case, convert a boolean to "yes" or "no", and much more!
