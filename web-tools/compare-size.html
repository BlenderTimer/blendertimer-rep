    Syntax:query

    Searches for objects with a name or tags that match or are similar to the specified name.


    Example:Human


    Syntax:artst:query

    Searches for objects with images created by the specified artist (or a similar artist).


    Example:artst:Daniel Roberts


    Syntax:id:query

    Returns the object with the specified object ID.


    Example:id:123


    Syntax:category:query

    Searches for objects in the specified category (or a similar category). (searches all 3 category levels)


    Example:category:planets


    Syntax:width:query

    Searches for objects with the specified width (or a similar width). (searches in kilometers)


    Example:width:0.01


    Syntax:length:query

    Searches for objects with the specified length (or a similar length). (searches in kilometers)


    Example:length:0.02


    Syntax:height:query

    Searches for objects with the specified height (or a similar height). (searches in kilometers)


    Example:height:0.03


    Syntax:date:query

    Searches for objects with the specified last-updated date (or a similar date). (format: yyyy/mm/dd)


    Example:height:2023/01/23


    Syntax:originaldate:query

    Searches for objects with the specified original date (or a similar date). (format: yyyy/mm/dd)


    Example:height:2023/01/23
